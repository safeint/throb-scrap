# 说明
学习Go的练手结果，目的是整理存在nas上的小黄们，方便在jellyfin中展示

# 效果
* 遍历指定目录下的所有.mp4视频文件
* 视频文件归档到{actor}/{code}的目录下
* 从javbus收集信息
* 下载花絮及海报
* 生成.nfo文件
